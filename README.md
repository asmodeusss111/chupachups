# Голосование за Чупачупса 🍭

Готовый статический мини-сайт для GitHub Pages.

## Что есть

- адаптивный красивый дизайн;
- кнопка «Нет», которая убегает от мыши и касаний;
- кнопка «Да» с конфетти;
- общий счётчик голосов через бесплатный Count API;
- защита от повторного голоса в том же браузере через `localStorage`;
- резервный локальный режим, если API временно недоступен.

## Быстрый запуск

Просто откройте `index.html` в браузере.

## Публикация на GitHub Pages

1. Создайте новый публичный репозиторий на GitHub.
2. Загрузите в корень репозитория все файлы из этой папки.
3. Откройте **Settings → Pages**.
4. В разделе **Build and deployment** выберите **Deploy from a branch**.
5. Выберите ветку **main**, папку **/(root)** и нажмите **Save**.
6. После публикации ссылка появится в этом же разделе.

## Сброс общей статистики

Откройте `script.js` и измените значение `COUNTER_KEY` на новую уникальную строку. Например:

```js
const COUNTER_KEY = "my-new-vote-counter-2026-random-123";
```

Важно: это развлекательный публичный счётчик. Его ключ виден в исходном коде, поэтому он не подходит для серьёзных выборов или защищённой аналитики.
